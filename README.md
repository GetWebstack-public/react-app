# GWS Playground Frontend

A React-based playground application for testing and experimentation.

## Quick Start

From the project root directory:

```bash
gws up
```

This will build and deploy the frontend to a local Kubernetes cluster along with all other services defined in the workspace.

## GWS Commands

- `gws init` - Initialize the workspace
- `gws status` - Check the status of all services
- `gws logs` - View logs for all services

## Tech Stack

- React 18
- React Scripts 5.0
- Testing Library
